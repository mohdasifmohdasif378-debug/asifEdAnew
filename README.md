# AsifEdA AI — Smart Exam Preparation Platform
An AI-powered educational platform for competitive exam preparation (UPSC, SSC, NDA, NEET, JEE, JKSSB).

## Features
- 🔐 Secure Authentication (JWT, bcrypt)
- 🤖 AI Chat (Anthropic Claude)
- 📱 Responsive Design
- 🐳 Docker Ready

## Quick Start
1. Clone the repo
2. cp .env.example .env (add your keys)
3. docker-compose up --build
4. Open http://localhost:3000
